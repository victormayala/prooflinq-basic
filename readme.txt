== Description ==

Prooflinq Basic is a WordPress plugin designed to streamline the feedback process for web designers, developers, and creative professionals. It provides an intuitive interface for clients to submit visual feedback directly on your website.

= Key Features =

* **Visual Feedback Collection**: Allow clients to point and click directly on your website to provide feedback
* **Interactive Feedback Button**: Easily accessible feedback button for users
* **User Authorization System**: Control who can provide feedback
* **Automated Log Management**: Keep track of all feedback activities
* **Admin Dashboard Integration**: Manage feedback through WordPress admin
* **Responsive Design**: Works seamlessly on all devices
* **Screenshot Capabilities**: Powered by html2canvas for accurate visual feedback

== Installation ==

1. Download the plugin from WordPress.org or your provided source
2. Upload the plugin files to the `/wp-content/plugins/prooflinq-basic` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure the plugin settings in the WordPress admin area

= Requirements =

* WordPress (latest version recommended)
* PHP 5.6 or higher
* Modern web browser with JavaScript enabled

== Frequently Asked Questions ==

= Is this plugin free? =

Yes, Prooflinq Basic is completely free. However, there's a premium version available with additional features.

= Can I upgrade to Premium later? =

Yes, you can easily upgrade to the Premium version at any time. Your existing feedback and settings will be preserved.

= How do I authorize users to provide feedback? =

User authorization can be managed through the plugin settings in your WordPress admin panel.

= Does this work with any WordPress theme? =

Yes, Prooflinq Basic is designed to work with any properly coded WordPress theme.

= How are the screenshots captured? =

We use html2canvas, a reliable JavaScript library, to capture accurate screenshots of your web pages.
